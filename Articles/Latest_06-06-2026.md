## 4. Ideogram 4.0: 9.3B Open-Weight Text-to-Image Model Tops Design Benchmarks

**Source:** [GitHub](https://github.com/ideogram-oss/ideogram4) | [Hugging Face](https://huggingface.co/ideogram-ai/ideogram-4-nf4)

Ideogram released Ideogram 4.0 on June 3, 2026 — its first open-weight image generation model, built from scratch as a 9.3B-parameter single-stream Diffusion Transformer (DiT). The model introduces a structured JSON prompting interface with explicit bounding-box layout and color-palette controls, best-in-class multilingual text rendering, and native 2K resolution output. It immediately claimed first place among open-weight models on the DesignArena leaderboard and ninth overall in the broader text-to-image arena. The inference code is Apache 2.0; model weights are gated on Hugging Face under a non-commercial license, requiring a paid license for production deployment.
