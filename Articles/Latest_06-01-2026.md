## 3. 'BadHost' (CVE-2026-48710): Critical Starlette Flaw Threatens Millions of AI Agents

**Source:** [OSTIF](https://ostif.org/disclosing-the-badhost-vulnerability-in-starlette/) | [SC Media](https://www.scworld.com/brief/high-severity-starlette-vulnerability-badhost-could-expose-sensitive-data)

A critical authentication bypass vulnerability dubbed "BadHost" (CVE-2026-48710) has been disclosed in Starlette, the Python ASGI framework underpinning FastAPI, vLLM, LiteLLM, MCP servers, and countless AI agent harnesses. The flaw allows attackers to inject characters (/, ?, #) into the HTTP Host header, shifting path boundaries so that `request.url.path` returns a different value than what the server actually routed — effectively bypassing auth middleware. With Starlette logging 325 million weekly downloads, the blast radius is enormous. The bug is patched in version 1.0.1, but widely-deployed older versions remain vulnerable.

## 5. Groupon Cuts 400 Jobs in AI-Native Pivot Under "Project Foundry"
